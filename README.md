# Employee Management Dashboard (React.js)

##  Project Overview
This is a React.js Employee Management Dashboard built as part of a technical assignment.  
The application allows authenticated users to manage employees with full CRUD functionality, search, filtering, and printing support.

Authentication is mock-based and data persistence is handled using browser LocalStorage.

---

## Features

### Authentication
- Login page with mock authentication
- Protected dashboard route
- Redirect to login if unauthenticated

### Dashboard Summary
- Total employee count
- Active / Inactive status tracking

### Employee Management
- Add new employees
- Edit existing employees
- Delete employees (with confirmation)
- Upload profile image with preview
- Active / Inactive toggle
- Print employee list

### Search & Filters
- Search by employee name
- Filter by gender
- Filter by active/inactive status
- Combined filters supported

### UI / UX
- Clean and modern layout
- Responsive table
- Empty state handling
- Image previews before saving

---

##  Tech Stack

- React.js (JavaScript)
- React Router DOM
- HTML5 / CSS3
- LocalStorage (Mock API)
- custom CSS
